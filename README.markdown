# IPLocator

### Locates the IP Country.

## usage

To run the application just type 
   
      java -jar /path/to/IPLocator.jar

or 

      java -jar /path/to/IPLocator.jar < file.txt

the file.txt is a file that can contain any number of ip addresses
for exaple :

     200.35.233.23
     189.35.1.3
     193.200.1.4
